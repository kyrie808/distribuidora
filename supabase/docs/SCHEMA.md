# Database Schema - Mont Distribuidora

## Overview

This document provides the detailed schema of the PostgreSQL database hosted on Supabase for the Mont Distribuidora project.

## Tables and Columns

### `produtos`
Main product catalog.
- `id`: uuid (Primary Key)
- `nome`: text
- `descricao`: text
- `preco`: numeric
- `categoria`: text
- `estoque`: numeric
- `imagem_url`: text
- `created_at`: timestamptz

### `vendas`
Sales transactions.
- `id`: uuid (Primary Key)
- `data`: timestamptz
- `cliente_id`: uuid (FK -> contatos)
- `total`: numeric
- `status`: text
- `origem`: text (e.g., 'catalogo', 'pdv')

### `contatos`
Customers and Suppliers.
- `id`: uuid (Primary Key)
- `nome`: text
- `email`: text
- `telefone`: text
- `tipo`: text ('CLIENTE', 'FORNECEDOR')

### `purchase_orders`
Orders sent to suppliers.
- `id`: uuid (Primary Key)
- `fornecedor_id`: uuid (FK -> contatos)
- `data`: timestamptz
- `status`: text

### `purchase_order_items`
Line items for purchase orders.
- `id`: uuid (Primary Key)
- `purchase_order_id`: uuid (FK -> purchase_orders)
- `produto_id`: uuid (FK -> produtos)
- `quantidade`: numeric
- `preco_unitario`: numeric

### `cat_pedidos` / `cat_itens_pedido`
Catalog-specific order management (Sync system).

## Relationships (Foreign Keys)

- `vendas.cliente_id` -> `contatos.id`
- `purchase_orders.fornecedor_id` -> `contatos.id`
- `purchase_order_items.purchase_order_id` -> `purchase_orders.id`
- `purchase_order_items.produto_id` -> `produtos.id`
