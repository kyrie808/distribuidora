# Epic: Security & Stability Foundation

## Description
This epic focuses on resolving the most critical technical debt identified during the discovery phase. The goal is to secure the database layer, establish a testing culture, and ensure data integrity through automation.

## Goals
- [ ] 100% RLS coverage for core public tables.
- [ ] Baseline test runner (Vitest) and CI integration.
- [ ] Automated inventory sync triggers.

## Related Stories
- [ ] **Story 01**: Implement RLS Policies for Sales/CRM.
- [ ] **Story 02**: Setup Testing Framework (Vitest/Playwright).
- [ ] **Story 03**: Database-level Inventory Triggers.
- [ ] **Story 04**: Implement Soft Deletes for Key Records.

## Business Value
Reduces the risk of data leaks and regressions, ensuring a professional and stable platform for business users.
